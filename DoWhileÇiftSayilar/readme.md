# Çift Sayılar Programı

Bu Java programı, kullanıcıdan aldığı iki sayı arasındaki çift sayıları ekrana yazdırır.

## Nasıl Çalışır?

Program, kullanıcıdan iki sayı alarak başlar. İlk olarak, kullanıcıya birinci sayıyı girmesi istenir. Ardından, kullanıcıdan ikinci sayı istenir.

Sonra, bir döngü kullanarak ilk sayıdan başlayıp ikinci sayıya kadar olan tüm sayıları kontrol eder. Eğer bir sayı çift ise, yani 2'ye bölündüğünde kalanı sıfır ise, o sayı ekrana yazdırılır.
